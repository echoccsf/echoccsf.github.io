My Coding Tips

Optimizing Images for the Web - I used the JPEG-Optimizer to optimize the image on my front page. The image was originally 540 by 540 pixels, and it's now 400 by 400 pixels. The original file size was 51 kb, and it's now 27.5 kb. The compression level was set at 65%, while the photo width was set at 400 pixels. This represents a 46% reduction in file size.
